Working Principle

How our Smart Home Automation System operates

1

Sensor Data Collection

The system continuously monitors environmental conditions using DHT11 (temperature/humidity) and LDR (light) sensors. This data is processed by the ESP32 microcontroller as the brain of the system to recieve, processe and send the command to control actuators.

2

Automated Decision Making

Based on predefined thresholds, the system automatically controls devices:

  • Lights turn ON when LDR value < 950, OFF when > 1050
  • Fan activates when temperature exceeds the maximum temperature
  • Windows open when light level > 2000 lux when is dark it closed
3

Security System Operation

The keypad allows users to enter a PIN had been set to unlock the door. Correct PIN activates the solenoid lock and status LEDs, while incorrect attempts trigger the buzzer and status of locked led.

4

MQTT Communication

The system connects to Adafruit IO via MQTT protocol to publish sensor data and receive control commands as dashboard, enabling remote monitoring and control from anywhere when is needed.

5

Manual Override

All automated functions can be manually controlled either through physical interfaces (keypad) or remotely via MQTT commands to Adafruit IO.

Project Components

Hardware and software components used in our Smart Home Automation System

ESP32 Microcontroller

The brain of our system, handling all informations from sensor,then processe them, and outputing them to control system with WiFi capabilities through dashboard.

DHT11 Sensor

(Digital humidity and temperature) sensor; it measures temperature and humidity for climate control and referring to automatic fan on and off to maintaining weater condition.

LDR Light Sensor

(Light Dependent Resistor) sensor;it detects ambient light levels for automatic lighting control based on light intensity to switch on and off automatically.

4x3 Matrix Keypad

Provides secure access control with password-based door unlocking functionality and help to input password to unlock door.

Relay Modules

Controls high-power devices like lights, fans, and solenoid locks safely with low-voltage signals.

Servo Motor

Controls window opening and closing based on light levels and automation rules and it moves in 180 angle .

Solenoid Lock

Electromagnetic lock for door security system, activated when correct password is entered via keypad.

12V DC Fan

Cooling fan controlled by relay module, automatically turns on when temperature exceeds desired level for climate controlling fan on and off.

Lamp

Lighting device controlled by relay module, automatically turns on/off based on ambient light levels detected by LDR sensor.

Jumper Wires

Connection wires used for interconnecting all components on the breadboard and establishing electrical connections between ESP32 and peripherals.

Breadboard

Used to connect components through circuit.

Resistors

Resistors are electronic components that oppose the flow of current through the conductor so we used them to protect LEDs and to make voltage divider a long side LDR sensor.

Light Emitting Diodes(LEDs)

Used to indicate the state on the door one for incorrect pass word and other one for correct password.

Buzzer

Audio alert device that sounds when incorrect password is entered or for system notifications and alarms.

// Automatic lighting control based on LDR sensor
void handleLighting() {
    if (!autoLighting) return;

    int lightLevel = analogRead(LDR_PIN);

    if (lightLevel < LIGHT_ON_THRESHOLD) {
        if (!lampState) {
            digitalWrite(RELAY1_PIN, HIGH);
            lampState = true;
            Serial.println("Lamp turned ON (dark)");
        }
    } else if (lightLevel > LIGHT_OFF_THRESHOLD) {
        if (lampState) {
            digitalWrite(RELAY1_PIN, LOW);
            lampState = false;
            Serial.println("Lamp turned OFF (bright)");
        }
    }
}

// Climate control based on temperature
void handleClimate() {
    if (!autoClimate) return;
    
    float temp = dht11.readTemperature();
    if (temp == -1) {
        Serial.println("Failed to read temperature");
        return;
    }
    
    if (temp > 28) {
        digitalWrite(RELAY2_PIN, HIGH);
    } else {
        digitalWrite(RELAY2_PIN, LOW);
    }
}

// Security system with keypad
void checkPassword() {
    Serial.println("Checking password: " + input);
    
    if (input == passwordCode) {
        digitalWrite(UNLOCK_LED, HIGH);
        digitalWrite(LOCK_LED, LOW);
        digitalWrite(RELAY3_PIN, HIGH); // Unlock solenoid
        digitalWrite(BUZZER_PIN, LOW);
        doorLocked = false;
        Serial.println("Access granted");
    } else {
        digitalWrite(BUZZER_PIN, HIGH);
        delay(1000);
        digitalWrite(BUZZER_PIN, LOW);
        Serial.println("Access denied");
    }
    input = "";
}

// MQTT connection handler
void MQTT_connect() {
    int8_t ret;

    if (mqtt.connected()) {
        return;
    }

    Serial.print("Connecting to MQTT... ");

    uint8_t retries = 3;
    while ((ret = mqtt.connect()) != 0) { 
        Serial.println(mqtt.connectErrorString(ret));
        Serial.println("Retrying MQTT connection in 5 seconds...");
        mqtt.disconnect();
        delay(5000); 
        retries--;
        if (retries == 0) {
            Serial.println("MQTT connection failed permanently!");
            return;
        }
    }
    Serial.println("MQTT Connected!");
}
